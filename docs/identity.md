# kafkax 身份

## 我是谁

`kafkax` 是 FoundationX 的 **Kafka 存储扩展模块**，提供 Kafka 客户端封装和标准消息操作接口。

## 我做什么

- Kafka 生产者/消费者接口封装
- 消息序列化/反序列化
- Topic 管理和连接池

## 我不做什么

- 不是业务消息处理器 — 业务消费逻辑由调用方定义
- 不是模板源 — 模板生成属于 xlib-standard
- 不依赖其他存储模块 — 存储扩展之间平级

## 宪法合规

| 条款 | 遵循方式 |
|------|----------|
| §3.3 | 存储扩展，可依赖 kernel + observex (interface-only) |
| §3.4 | 不依赖 configx、业务域、其他存储扩展 |
| §1 P13 | 存储扩展之间平级协作 |
